# Línea de Tiempo DAGTech

Este proyecto es una línea de tiempo interactiva que visualiza la historia de innovación y los hitos más importantes del Grupo Domingo Alonso (DAGTech) desde el año 2000 hasta el 2025.

## Objetivo

El objetivo principal es mostrar de manera clara y visual la evolución tecnológica, de productos y organizacional del grupo, permitiendo filtrar por categorías y destacar los hitos más relevantes.

## Características

- **Visualización Cronológica**: Muestra hitos ordenados por año.
- **Filtros por Categoría**: Permite filtrar entre hitos de AIDA (Tecnología), DTO (Transformación Digital) y Negocio.
- **Gestión de Datos**: Incluye un panel para editar, importar y exportar los datos de los hitos en formato Markdown.
- **Modos de Visualización**: Soporte para vista compacta y filtrado de hitos importantes.

## Tecnologías

- HTML5
- CSS3 (Diseño responsivo y moderno)
- JavaScript (Lógica de la aplicación y gestión de datos)
